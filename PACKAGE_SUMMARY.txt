QUICK START
================================================================================

1. EXTRACT
   tar -xzf ove-vm-migration-toolkit-0.9.0.tar.gz
   cd ove-vm-migration-toolkit

2. SETUP
   chmod +x scripts/*.sh

3. VERIFY PREREQUISITES
   oc whoami                    # Check OpenShift connection
   virtctl version              # Verify virtctl installed

4. RUN
   cd scripts
   ./orchestrate-migration.sh

5. FOLLOW MENU
   - Option 1: Assess VMs
   - Option 2: Create migration list
   - Edit vm-migration-list.txt
   - Option 3: Validate list
   - Option 11: Run full migration

KEY FEATURES
================================================================================

✓ Selective VM migration (not bulk namespace operations)
✓ Dynamic namespace prompts (no hardcoded values)
✓ PVC cloning using CSI dataSource (PowerFlex support)
✓ Comprehensive validation at every stage
✓ Real-time progress monitoring
✓ Safe cleanup with confirmations
✓ Detailed audit reports
✓ Complete rollback procedures

================================================================================
REQUIREMENTS
================================================================================

PLATFORM:
  - OpenShift 4.10+ with OpenShift Virtualization
  - Dell PowerFlex CSI driver (or compatible CSI with cloning support)

CLIENT TOOLS:
  - oc CLI (matching cluster version)
  - virtctl CLI
  - bash 4.0+
  - jq (for JSON parsing)

PERMISSIONS:
  - Admin access to source namespace
  - Admin access to target namespace (or ability to create)
  - Namespace create permission (if target doesn't exist)
